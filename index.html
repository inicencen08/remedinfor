APA ITU DOMAIN?

Domain adalah Sebuah nama unik yang menjadi alamat dari suatu website.

Jenis jenis domain berdasarkan ekstensi nya =
- CCTLD : Country Code Top Level Domain
.Id , .Us , .Jp , .Sg
- GTLD : Generic Top Level Domain
.Edu , .Org , .Net , .Com
- NGTLD : New General Top level Domain
.Xyz , .Museum , .Art , .Music

Fungsi Domain :
- Untuk mempermudah saat mengakses alamat.
- mengubah deretan alamat IP ke dalam sebuah istilah atau nama yang bisa dilihat dan diingat dengan mudah.
Misalnya, alamat IP yang dimiliki oleh domain biznetgio.com ialah 128.250.17.47.
- Meningkatkan Kredibilitas: Domain yang menggunakan ekstensi profesional seperti “.com” atau “.org” akan meningkatkan kredibilitas situs web di mata pengunjung, Kredibilitas ini terutama penting untuk situs bisnis atau situs resmi lainnya.

Perbedaan DOMAIN vs URL (Universal Resource Locator) :
-URL terdiri dari protekol , domain dan path (jalur)
protokol menunjukan apakah website memilikki serifikat SSL . URL hanya memiliki path kalau mengarahkan pengunjung ke halaman.
-URL lebih lengkap link nya.
-Domain adalah nama situs, sementara URL akan mengarahkan pengguna ke salah satu halaman dalam situs tersebut

  • Cara Kerja Domain :

User

Ketika user mengakses nama domain melalui browser. Server akan menjangkau jaringan server global yang membentuk DNS ( sitem nama domain)

IP/DNS Server

DNS server akan mencari alamat ip yang terkait dengan nama domain tersebut

DNS Server

DNS Server yang memiliki informasi tentang alamat IP Domain lalu memberikan hasilnya ke browser.

User

Browser akan meminta data tentang website dari server hosting domain setelah server.

Hosting


Hosting adalah layanan yang menyimpan situs web atau aplikasi web Anda dan membuatnya mudah diakses di berbagai perangkat seperti desktop, seluler, dan tablet.
Fungsi Hosting adalah sebagai tempat penyimpanan file dan data website,yang memungkinkan pengunjung untuk mengakses informasi tersebut secara online.

Cara Kerja Hosting :
-File situs web disimpan di server hosting.
- File website seperti gambar, video, script, database dan konten lainnya akan disimpan di server hosting.
-Permintaan yang dikirim ke server hosting , Permintaan dari pengunjung akan dikirim ke server hosting yang akan memproses permintaan tersebut.
-Server Hosting Mengirim File ke Browser, Setelah permintaan diterima server hosting akan mengirimkan file yang diminta seperti file HTML, gambar, atau konten lainnya ke browser pengunjung.
-Pengunjung Dapat Mengakses Situs Web, Setelah browser menerima file situs web dari server hosting, pengunjung dapat mengakses situs web dengan melihat halaman web yang terbentuk di browser.

Jenis-Jenis Hosting :
-Shared Hosting : jenis hosting paling murah dan paling mudah sehingga cocok untuk bisnis kecil dan website pribadi yang tidak memerlukan konfigurasi tingkat lanjut atau bandwidth yang tinggi.
-VPS Hosting : Virtual Private Server atau VPS adalah pilihan yang tepat bagi Anda yang ingin membuat website skala medium, toko online, dan blog besar yang menargetkan pertumbuhan pesat terkait jumlah pengunjung
-Cloud Hosting : pilihan tepat bagi bisnis yang punya banyak website dan website berskala besar seperti toko online.
-WordPress Hosting : salah satu tipe CMS hosting yang servernya dioptimalkan untuk WordPress agar loadingnya lebih cepat dan minim masalah.

Membuat Halaman Website

Hosting berfungsi untuk menciptakan sebuah situs dengan banyak banyak informasi dan dokumen terkait pengguna atau bisnis di dalamnya. Dengan menggunakan paket hosting, maka halaman situs anda akan ditampilkan di search engine supaya dapat diakses oleh pengunjung melalui internet. .

Mengembangkan Situs Jual Beli Online

Apabila Anda ingin membuat website toko online, maka Anda harus menggunakan hosting agar informasi yang anda cantumkan dapat terindeks mesin pencari. Pengembangan website toko online berpotensi mendatangkan konsumen lebih besar dan cakupan luas.

Membuat Email Profesional

hosting juga berfungsi untuk membuat email profesional dengan domain Anda sendiri. Tujuannya untuk meningkatkan kredibilitas bisnis anda. Di sini, Anda juga akan terlihat lebih profesional.

Mampu Melindungi Website dengan SSL

Hosting juga melindungi website dari serangan malware, virus, serangan DDOS atau spam. Maka, dengan adanya hosting, anda dapat memasang SSL (Secure Socket Layer) supaya situs anda lebih aman dan maksimal.
